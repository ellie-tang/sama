Dementia affects millions of individuals worldwide, causing two critical challenges: facial recognition deficits and speech impairements. This project introduces a novel AI framework integrated with augmented reality (AR) glasses to create a continously learning AI assistant for dementia patients in daily life that adapts to their needs. The glasses frontend uses a camera, microphone, speaker, and heads up display (HUD) integrated with the backend computing server for the following functions. First, the AI assistant continuously records and learns from the user's daily experiences to memorize new faces and contextual information, providing real time name and relationship prompts for recognized individuals. Second, for disordered speech, the system continuously trains itself on the user's voice patterns based on the user's daily speech data, enabling automatic speech recognition (ASR), multimodal regeneration, and normalized speech translation through a speaker. The ASR and facial recognition models continually finetune and evolve at night, aiming to improve communication and social confidence for patients.
